FAQ

What is this?

Kickoff is a scoping tool for analysts and ops professionals. When an executive asks a vague question ("why is retention down?"), the tool decomposes it into a structured analysis brief through a five-minute interview. The brief becomes a shareable permalink you send to the exec for scope alignment before you write any SQL.

Who is it for?

Analysts and operations pros at nonprofits and mid-market companies. If you receive 3-8 ad hoc data requests per week from executives, this tool exists to save you from doing the wrong analysis.

Is it free?

Yes. No signup, no API key, no account. Just paste a question and go. Rate limited to prevent abuse (10 briefs per IP per hour).

Do you store my data?

Every completed brief is stored so its permalink continues to work. No email, no name, no IP is collected. Treat the permalink as public — anyone with the URL can view the brief.

Can I use it for team briefs?

v1.0 is single-user only. Team workspaces, brief comments, and collaboration land in v2. Feature requests welcome — file an issue on GitHub.
